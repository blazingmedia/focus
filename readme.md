# Focus by BlazingMedia
User Stylesheets that simplify the experience on various websites

> We highly suggest you to use [Stylus](https://github.com/openstyles/stylus) instead of Stylish due to Privacy Reasons! Read more [here](https://robertheaton.com/2018/07/02/stylish-browser-extension-steals-your-internet-history/).

## Themes

### YouTube Music
![YouTube Music](/assets/ytmusic1.webp)

#### Changes
This Theme changes the overall color scheme of YouTube Music, removes some Spacing on the Frontpage and removes the YouTube Video Feed (The Video still plays, it's just hidden, it still uses up bandwidth!)
